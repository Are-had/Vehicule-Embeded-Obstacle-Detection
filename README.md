# PFE Stereo Vision 
# Démo PFE – Visualisation obstacles (Raid)

## Objectif
Démo rapide : génération d'une base CSV (lat, lon, type, volume, image)
et visualisation sur une carte via Streamlit.

## Installation
pip install -r requirements.txt

## Générer les données
python generate_demo_data.py

## Lancer l'application
streamlit run app.py
